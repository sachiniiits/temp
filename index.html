CS2 Match Analytics Pipeline

btp_cs2 ↗

An end to end behavioural analytics pipeline that ingests raw CS2 demo files from IEM Katowice and StarLadder Budapest Major 2025, then classifies each player's per round role, Entry, AWPer, Support, Lurker or IGL, across roughly 36,000 player rounds with no labelled data. It extracts 14 behavioural features per round and assigns roles with a KMeans, GMM and HDBSCAN ensemble aligned by the Hungarian algorithm.

Autonomous Multi-Agent Game Engine

AI_Game_Engine ↗

A five stage agentic pipeline, Research to Script to Logic to Coding to Validator, that turns a single prompt into a fully playable browser game. Each agent receives the full upstream context to avoid cross stage hallucination, and a Validator agent runs a self correction loop that statically checks the output, guaranteeing a zero dependency game file on every run.
